## ✍️ Using the Editor

The editor uses the markdown format. This is a plain text style that adds formatting to your writing. Use symbols to create headings, lists, and links. You do not need to learn code. Simply type as you normally would.

*   **Headings**: Put a hash symbol (#) at the start of a line to create a heading.
*   **Bold**: Wrap text in two asterisks like \*\*this\*\* to make it bold.
*   **Lists**: Press the hyphen (-) key at the start of a line to make a list.
*   **Math**: Use dollar signs ($) to wrap your math formulas.
*   **Tabs**: Click the plus icon to add a new file.

**Is this program safe?**
Yes. The software uses a secure process to run on your system. It only accesses files you explicitly open within the app.

**Does this require an internet connection?**
No. You can write and save files while offline. You only need the internet to download the installation file.

**Can I move my files to other programs?**
Yes. All your files are saved in the standard markdown format. You can open them in any other text editor.

**How do I update the program?**
Download the latest version from the link above and run the installer again. The app replaces the old version with the new one while keeping your settings.

**What happens if the app crashes?**
The system creates background backups while you type. When you open the app again, it looks for these backups to recover your unsaved changes.
